------------
How to run:
------------
Code does not parse command line arugments instead functionality is controlled
by uncommenting out function calls.  To generate training sets uncomment out the 
GenerateTrainingSets() call.  To build and train the network uncomment out the 
buildAndTrainNet() method call.  To run an existing network uncomment out the 
RunExistingNetwork() method call.

----------------
What you need:
----------------
- Python 2.7:
   * Keras library 
   * numpy library 
   * lxml
   * scipy
   * pydub
   * there may be other dependencies on these libaries

- MinGW
